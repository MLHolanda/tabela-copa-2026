# Tabela Copa 2026

Projeto desenvolvido para estudo de HTML, CSS e JavaScript.

## Funcionalidades

* Simulação de partidas
* Atualização automática da classificação
* Pontos, vitórias, empates e derrotas
* Saldo de gols
* Tabela responsiva

## Tecnologias

* HTML5
* CSS3
* JavaScript

## Próximas versões

* Gols Pró (GP)
* Gols Contra (GC)
* LocalStorage
* Grupos completos da Copa 2026
* Fase eliminatória
